\subsection{(Scalar) GARCH and Multivariate GARCH}
Before I turn to the functional GARCH model, I will focus on the scalar version of the GARCH model, both in the univariate and multivariate cases.\\
The GARCH model, a generalization of the previously published ARCH model by \citet{Engle1982}, was introduced by \citet{Bollerslev1986}. (G)ARCH models fundamentally differentiate themselves from typical time series models, such as ARMA or VAR models. Instead of modeling the relationship between the observed values themselves and their previously observed values, the relationship between the squared observed values and their previous squared values is modeled. This is due to the following characteristics of financial time series data: Financial price data is generally not stationary but is more comparable to a random walk. Because of that, return data is typically used for modeling purposes. When examining return data, it can be seen that while return data itself displays almost no or small autocorrelations, squared or absolute return data does display significant autocorrelations. Furthermore, large fluctuations appear to occur in clusters that are recurrent but not periodic \citep{mandelbrot1963variation,taylor2011asset}. Therefore, (G)ARCH models try to model the relationship of squared returns.\\
An univariate GARCH(p,q) model can be described by its first two moments:
\begin{definition}[GARCH(p,q) Process]
    A process $(\varepsilon_t)$ is called a GARCH(p,q) process if its first two conditional moments exist and satisfy:
    \begin{enumerate}
        \item $\mathrm{E}(y_t \,|\, y_u,u<t)=0, \quad \forall t\in\mathbb{Z}$.
        \item There exist constants $\delta,\alpha_i,i=1,...,q$ and $\beta_j,j=1,...,p$ such that\\
        \begin{align}
            \sigma_t^2=\mathrm{Var}(y_t\,|\,y_u,u<t)=\delta+\sum_{i=0}^q\alpha_iy_{t-1}^2+\sum_{j=1}^p\beta_j\sigma_{t-1}^2
        \end{align}
    \end{enumerate}
\end{definition}
A more restrictive, but often useful definition of a GARCH process is a strong GARCH(p,q) process. A strong univariate GARCH(p,q) process can be represented as follows:
\begin{align}
    y_t &= \sigma_t\varepsilon_t \\
    \sigma_t^2 &= \delta + \sum_i^p \alpha_i y_{t-i}^2 + \sum_j^q \beta_j \sigma_{t-i}^2
\end{align}
where $(\varepsilon_t)$ is $\mathrm{i.i.d.}$. Additionally, it is often assumed that $E[\varepsilon_t|\varepsilon_j, j<t]=0$ and $\mathrm{Var}[\varepsilon_t]=1$.\\
I will focus on the strong GARCH(1,1) process since an equivalent functional version of this process is also used in \citet{Aue2017} as well as \citet{RICE20201023}.
The simplest specification of this model is the GARCH(1,1) process. Nevertheless, this specification is also one of the most widely used in practice.\\ % Add reference!!!
It can be shown that there exists a strictly stationary, non-anticipative, ergodic solution to the GARCH(1,1) process if the following condition is satisfied:
\begin{theorem}[Strict stationarity of the strong GARCH(1,1) process]
\label{stationarityscalarGARCH(1,1)}
    If
    \begin{align}
        -\infty\leq \gamma &:=\mathrm{E}\,\ \mathrm{log}\{\alpha\varepsilon_t^2+\beta\}<0,
    \end{align}
    then the infinite sum
    \begin{align}
        h_t &= \left[ 1+\sum_{i=0}^{\infty}a(\varepsilon_{t-1})...a(\varepsilon_{t-i}) \right]\delta; \quad a(z)=\alpha z^2+\beta
    \end{align}
    converges almost surely, and the process $(y_t)$ defined by $y_t=\sqrt{h_t}\varepsilon_t$ is the unique strictly stationary solution of model (2.7). This solution is non-anticipative and ergodic.
\end{theorem}
