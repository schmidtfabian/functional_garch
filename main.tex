\subsection{(Scalar) GARCH and Multivariate GARCH}
Before I turn to the functional GARCH model, I will focus on the scalar version of the GARCH model, both in the univariate and multivariate cases.\\
The GARCH model, a generalization of the previously published ARCH model by \citet{Engle1982}, was introduced by \citet{Bollerslev1986}. (G)ARCH models fundamentally differentiate themselves from typical time series models, such as ARMA or VAR models. Instead of modeling the relationship between the observed values themselves and their previously observed values, the relationship between the squared observed values and their previous squared values is modeled. This is due to the following characteristics of financial time series data. Return data itself displays almost no or small autocorrelations, squared or absolute return data, however, does display significant autocorrelations. Furthermore, large fluctuations appear to occur in clusters that are recurrent but not periodic \citep{mandelbrot1963variation,taylor2011asset}. (G)ARCH models are therefore a natural conclusion of these stylized facts.\\
An univariate GARCH(p,q) model can be described by its first two moments:
\begin{definition}[GARCH(p,q) Process]
    A process $(y_t)$ is called a GARCH(p,q) process if its first two conditional moments exist and satisfy:
    \begin{enumerate}
        \item $\mathrm{E}(y_t \,|\, y_u,u<t)=0, \quad \forall t\in\mathbb{Z}$.
        \item There exist constants $\delta,\alpha_i,i=1,...,q$ and $\beta_j,j=1,...,p$ such that\\
        \begin{align}
            \sigma_t^2=\mathrm{Var}(y_t\,|\,y_u,u<t)=\delta+\sum_{i=0}^q\alpha_iy_{t-i}^2+\sum_{j=1}^p\beta_j\sigma_{t-j}^2
        \end{align}
    \end{enumerate}
\end{definition}
A helpful representation of the GARCH(p,q) process can be obtained by defining an error term $\nu_t=y_t^2-\sigma_t^2$ and replacing $\sigma^2$ with $y^2+\nu$
\begin{align}
\label{ARMAstructure}
    y_t^2=\delta+\sum_{i=0}^r\overbrace{(\alpha_i+\beta_i)}^{\text{$\kappa_i$}}y_{t-i}^2+\nu_t-\sum_{j=1}^p\beta_j\nu_{t-j}=\delta+\sum_{i=0}^r\kappa_iy_{t-i}^2+\nu_t-\sum_{j=1}^p\beta_j\nu_{t-j}
\end{align}
where $r=\mathrm{max}(p,q)$ with $\alpha_i=0$ ($\beta_j=0$) if $i>q$ ($j>p$). From this representation, it can be seen that a GARCH(p,q) process follows an ARMA(r,p)-like structure for the squared values $y_t^2$. Note that $(\nu_t)$ is not i.i.d. and therefore cannot be used to examine the existence of a stationary solution of the GARCH(p,q) process.\\
A more restrictive yet often useful definition of a GARCH process is that of a strong GARCH(p,q) process. A strong univariate GARCH(p,q) process can be represented as follows:
\begin{gather}
    y_t = \sigma_t\varepsilon_t \\
    \sigma_t^2 = \delta + \sum_i^p \alpha_i y_{t-i}^2 + \sum_j^q \beta_j \sigma_{t-i}^2
    \label{Eq2StrongGARCH}
\end{gather}
where $(\varepsilon_t)$ is $\mathrm{i.i.d.}$ Additionally, it is often assumed that $E[\varepsilon_t|\varepsilon_j, j<t]=0$ and $\mathrm{Var}[\varepsilon_t]=1$.\\
I will focus on the strong GARCH(1,1) process since an equivalent functional version of this process is also used in \citet{Aue2017} as well as \citet{RICE20201023}.
The strong GARCH(1,1) process is the simplest specification of the strong GARCH(p,q) processes. Nevertheless, this specification is also one of the most widely used in practice.\\ % Add reference!!!
Before listing the condition for a stationary solution to the strong GARCH(1,1) process to exist, note that the second equation of the strong GARCH(1,1) process can be rewritten as follows:
\begin{align}
    \sigma_t^2 = \delta + a(\varepsilon_{t-1}) \sigma_{t-i}^2, \quad \text{where }a(\varepsilon_t)=\alpha\varepsilon_t^2+\beta
\end{align}
From this representation, it can be seen that the volatility equation of the GARCH process follows an AR-structure with a random coefficient.\\
It can be shown that there exists a strictly stationary, non-anticipative, ergodic solution to the strong GARCH(1,1) process if the following condition is satisfied:
\begin{theorem}[Strict stationarity of the strong GARCH(1,1) process]
\label{stationarityscalarGARCH(1,1)}
    If
    \begin{align}
        -\infty\leq \gamma &:=\mathrm{E}\,\ \mathrm{log}\{\overbrace{\alpha\varepsilon_t^2+\beta}^{a(\varepsilon_t)}\}<0,
    \end{align}
    then the infinite sum
    \begin{align}
        h_t &= \left[ 1+\sum_{i=0}^{\infty}a(\varepsilon_{t-1})...a(\varepsilon_{t-i}) \right]\delta; \quad a(z)=\alpha z^2+\beta
    \end{align}
    converges almost surely, and the process $(y_t)$ defined by $y_t=\sqrt{h_t}\varepsilon_t$ is the unique strictly stationary solution of model (2.7). This solution is non-anticipative and ergodic.
\end{theorem}
To prove \ref{stationarityscalarGARCH(1,1)}, note that Equation (\ref{Eq2StrongGARCH}) can be solved recursively to attain $h_t$. Showing that $h_t$ converges involves an application of the Cauchy Root Test. Uniqueness then follows by contradiction by assuming the existence of another solution and showing that it must coincide with the original solution. The complete proof can be found in the Appendix.\\ % Maybe try using Cleveref here if there is time.

A functional GARCH process is defined as follows:
\begin{definition}[Functional GARCH(p,q) Process]
\label{fGARCH(p,q)}
A sequence of random functions $(y_i(t))$ is called a functional GARCH process if it satisfies the following equations:
    \begin{gather}
        y_i(t) = \sigma_i(t)\varepsilon_i(t)\\
        \sigma_i^2(t) = \delta(t)+\sum_{j=1}^p\alpha_jy_{i-j}^2(t)+\sum_{k=1}^q\beta_k\sigma_{i-k}^2(t)
    \end{gather}
\end{definition}
Here, $\delta(t)$ is a non-negative function and $(\alpha_j)_{j\in (1,...,p)}$ and $(\beta_k)_{k \in (1,...,q)}$ are both integral operators $\alpha_jx(t)=\int \alpha_j(t,s)x(s)ds$ ($\beta_jx(t)=\int \beta_j(t,s)x(s)ds$) that map non-negative functions to non-negative functions. $x$ here denotes an arbitrary element in $L^2[0,1]$. By convention, I restrict the intraday time variable to $t\in[0,1]$. This is without loss of generality, since data that does not follow this convention can easily be converted into this format. $\int_0^1$ is therefore shortened to $\int$. The second time variable $i$ denotes the trading day. $(\varepsilon_i(t))_{i\in\mathbb{Z}}$ are i.i.d. functions.\\
From Definition \ref{fGARCH(p,q)}, it is easily seen that the Functional GARCH(p,q) process follows a similar functional structure to the strong (scalar) GARCH process. As a consequence, Definition \ref{fGARCH(p,q)} can also be rewritten into a similar ARMA-like structure like the scalar version in Equation (\ref{ARMAstructure}).\\
\\

        \sigma_i^2(t) = \delta(t)+\alpha y_{i-j}^2(t)+\beta\sigma_{i-k}^2(t)
\end{gather}
